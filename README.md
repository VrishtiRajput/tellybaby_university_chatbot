# 📖 About

**TellyBaby** is an AI-powered University Assistant that helps students manage their entire academic life from a single interface.

Instead of searching through ERP portals or remembering schedules manually, simply ask TellyBaby.

Whether you're online or offline, TellyBaby has your back.

---

# ✨ Features

## 📅 Smart Timetable
- View today's schedule
- Check tomorrow's classes
- Find your next lecture
- Upload custom timetable files

---

## 📊 Attendance Tracker

- View attendance percentage
- Subject-wise attendance
- Overall attendance calculation
- Upload updated attendance sheet

---

## 📝 Exam Dashboard

- Upcoming exams
- Marks history
- Exam schedule
- Days remaining before exams

---

## ⏰ Reminder System

- Create reminders
- Mark reminders complete

- Stay organized

# ⚡ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/TellyBaby.git
```

Go inside

```bash
cd TellyBaby
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```
